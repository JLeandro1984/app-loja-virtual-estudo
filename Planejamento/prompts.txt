 Prompt parte 1

 # Objetivo

Desenvolver um catálogo digital Premium totalmente responsivo utilizando apenas HTML5, CSS3 e JavaScript ES6+, sem backend e sem banco de dados.

O projeto deve possuir aparência profissional, código limpo, arquitetura modular, alta organização e ser preparado para uma futura migração para uma API REST, mantendo o menor impacto possível na estrutura do frontend.

O resultado deve transmitir qualidade semelhante a grandes e-commerces modernos.

---

# Objetivos do projeto

O catálogo servirá para divulgar produtos.

O visitante poderá:

- visualizar os produtos
- pesquisar produtos
- filtrar produtos
- ordenar produtos
- visualizar detalhes
- selecionar vários produtos
- alterar quantidade
- visualizar o total da compra
- enviar automaticamente sua lista de interesse para WhatsApp
- acessar o Instagram da loja
- compartilhar produtos

Não haverá pagamento online.

O objetivo é apenas facilitar o contato comercial.
